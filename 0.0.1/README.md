Lets you call GML functions from your Web Assembly (GX) GameMaker game 

# Features

- Wrappers for various GameMaker functions
- Support for creating and managing dynamic data from JavaScript, such as Buffers, Surfaces, and Data Structures
- TypeScript typings for various GameMaker types

---

# Why was GX Types made?

I first messed with buffers directly for `wasm-bridge`, overwriting game memory directly from the webpage. Then, TabularElf found the `__js_get_gml` function in WASM exports. This is a natural expansion on that.
